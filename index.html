EBRAINS WebComponents

Youtube Video Privacy Overlay

yt-video

Displays an overlay warning the user about Youtube/Google privacy if s.he loads/watches the video. Clicking on "watch video" button makes the embedded iframe created in the document

Attributes

url*
url embedded url video (e.g: https://www.youtube.com/embed/lyQfZZiBQVk?autoplay=1)
cover
image url that will be used as a cover (e.g: https://strapi-prod.sos-ch-dk-2.exo.io/ebrains_ssd_call_ebrains_website_3a68136980.jpg)

Styles

CSS custom properties

--yt-video-overlay-fgcolor
Overlay text color
--yt-video-overlay-bgcolor
Overlay background color: if opaque, the cover is not visible
--yt-video-overlay-gradient-from
Overlay background gradient start color: if opaque, the cover is not visible
--yt-video-overlay-gradient-to
Overlay background gradient to color: if opaque, the cover is not visible

Usage

Reference component

			
<script src="http://ebrains.bind.run/components/yt-video.js"></script>

Use in HTML

			
<yt-video
	url="https://www.youtube.com/embed/lyQfZZiBQVk?autoplay=1"
	cover="https://strapi-prod.sos-ch-dk-2.exo.io/ebrains_ssd_call_ebrains_website_3a68136980.jpg">
</yt-video>
You have to style yt-video on your side. If you don't define css width and height properties, component should be not visible.
